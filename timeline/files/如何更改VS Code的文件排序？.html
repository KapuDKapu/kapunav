visual-studio-code – 如何更改VS Code的文件排序？

时间  2019-03-09

标签 visual-studio-code 栏目 Visual Studio

 

在我的项目中,我有文件标记为day1.txt,day2.txt,day3.txt等.

 

现在在我的普通文件系统中,它们按正确的顺序列出：

 

day1.txt
day2.txt
....
day10.txt
day11.txt

在VS Code中,它们的列表如下：

 

day1.txt
day10.txt
day11.txt
day2.txt
day3.txt

有什么地方可以改变这个吗？这让我很生气.

我给了设置一个搜索,我没有看到任何明显的参数改变会影响这一点.

是的,有办法.

 

在您的设置(文件→首选项→设置)中搜索：explorer.sortOrder

设置的左侧会出现一个小笔.选择要对文件进行排序的方式.有五种选择：

>默认
>混合
>首先是文件
>类型
>修改
